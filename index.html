Vatman

A gem that checks VAT numbers against the Europa web service

View the Project on GitHub neilmiddleton/Vatman

  • Download ZIP File

vatman is a library for checking VAT numbers against the Europa webservice. This is the ONLY way to check if a VAT number is genuine and in use.

Usage

vat_number =  Vatman::Check.new('GB', 802925148)

vat_number.valid? => true
vat_number.name => 'MONOCHROME LIMITED'
vat_number.address => 'SUITE K\nTHE PAVILLIONS\n1 WESTERN ROAD\nEPSOM\nSURREY\nKT17 1JG'

Installation

gem install vatman

and then

require 'vatman'
